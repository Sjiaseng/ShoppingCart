# Description
Springboot and React Project Based On: https://www.youtube.com/watch?v=-Fe0zk-F4OA&t=18603s

## Setup

1. Clone Repo
```
git clone https://github.com/Sjiaseng/ShoppingCart.git
```

2. Setup Backend
```
cd myWebApp
```

3. Run Backend
```
mvn spring-boot:run
```

4. Setup Frontend
```
cd myWebApp2
```

5. Install Dependencies for Frontend
```
npm install
```

6. Run Frontend
```
npm run dev
```
